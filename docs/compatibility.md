| --- | --- |
| Baseline | Cocos Creator 3.8.8 |
| Intended range | Cocos Creator 3.8.x, subject to explicit validation |
| Production projects | Not supported during Alpha |

| Windows | Not yet claimed as validated |

兼容声明只覆盖实际完成验收的版本和平台。Issue 中出现的其他环境信息属于待验证报告，不自动扩大支持范围。
