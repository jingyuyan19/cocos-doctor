# Public Release Policy

公开仓库目前不发布 Alpha 安装包。任何未来公开 Release 必须同时满足：

1. 市场验证已给出明确的 Go 或经记录的公开发布决定；
2. 安装、升级、卸载和核心诊断链路针对目标 Creator 版本重新验收；
3. Release 资产与私有流水线中通过验收的字节完全一致；
4. 提供 SHA-256、兼容范围、许可、第三方声明和脱敏验收摘要；
5. 不包含商业源码、研究任务、参与者资料、密钥、本机绝对路径或未授权第三方资产；
6. Draft Release 经人工复核后才可公开；公开后不悄悄替换同版本资产。

GitHub Public Releases 不是付费下载门禁。商业版本的交付和授权将使用明确的外部分发渠道。
