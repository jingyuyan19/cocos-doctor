# Public Roadmap

路线会依据真实使用数据调整，不构成发布日期或功能承诺。

## Alpha validation

- 验证 Creator 3.8.8 安装与首次价值时间。
- 验证三类 UI 故障的证据是否可理解。
- 验证 Preview、Apply、复验和 Undo 的信任边界。
- 决定 Go、Iterate 或 No-go。

## Candidate follow-ups

- 更接近运行态的 Preview 证据采集。
- 稳定、可脱敏的诊断报告格式。
- 在不扩大写权限的前提下增加解释能力。
- 评估公开 EvidencePack / DiagnosisReport 协议与验证器。

在 Alpha 决策前，不以增加规则数量代替市场验证。
